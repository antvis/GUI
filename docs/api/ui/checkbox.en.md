<description>**optional** _number_</description>

Marker 标记，在 canvas 画布中 x 方向的的相对位置, 默认为 0.

#### y

<description>**optional** _number_</description>

Marker 标记，在 canvas 画布中 y 方向的的相对位置, 默认为 0.
